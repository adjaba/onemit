Creating One.MIT

The One.MIT team drew upon ingenuity, persistence, and creativity from colleagues across MIT. To start, they had to assemble the list of names by scouring resources—digital databases from recent years, old directories and lists from earlier eras—to compile the most comprehensive record to date of the MIT community’s members.

The names were then mapped over an image of the dome, with each letter assigned a value as a light or dark “pixel” according to the tone beneath it in the underlying image. An algorithm determined the global text size to maximally fill the printing space—a nonlinear optimization problem. The code was written in Mathematica and LaTeX. Each letter had to be turned into a graphical curve object and modified to adopt its light/dark characteristic.

Then, using photolithography and etch processes designed for micro- and nano-technology, the final image file was imprinted onto the silicon wafer you see here. The wafer was fabricated in the Microsystems Technology Laboratory on campus.
